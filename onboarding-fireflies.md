# ✅ Fireflies.ai Setup & Onboarding Checklist

> Get Fireflies.ai fully configured and extracting value from your first meeting. Estimated setup time: 15–20 minutes.

**Tool Profile & Full Details:** [Fireflies.ai on AI Central Resources](https://www.aicentralresources.com/tool/firefliesai)

---

## Phase 1: Account Setup

- [ ] Sign up at [fireflies.ai](https://fireflies.ai) (use work email for team features)
- [ ] Verify your email address
- [ ] Choose your plan (Free to start, upgrade when ready)
- [ ] Set your timezone (affects meeting timestamps)
- [ ] Add your name and profile photo

---

## Phase 2: Calendar & Meeting Integration

- [ ] Connect Google Calendar **or** Outlook Calendar
- [ ] Grant calendar read access so Fireflies can auto-join scheduled meetings
- [ ] Add Fred (the Fireflies bot) to your meeting platform:
  - [ ] **Zoom** — Install the Fireflies Zoom App from the Zoom marketplace
  - [ ] **Google Meet** — Enable via calendar integration (auto-joins)
  - [ ] **Microsoft Teams** — Add the Fireflies Teams app
  - [ ] **Webex / GoToMeeting / BlueJeans** — Add bot manually via meeting invite
- [ ] (Optional) Customize your Fireflies notetaker bot name
- [ ] Test with a dummy meeting to confirm the bot joins correctly

---

## Phase 3: Team & Collaboration Setup (Business+ Plans)

- [ ] Invite team members (Settings → Team → Invite)
- [ ] Set up shared workspaces or channels
- [ ] Configure who can access whose transcripts
- [ ] Set default sharing permissions (private / team / public)

---

## Phase 4: Integrations

- [ ] **Slack** — Connect to auto-post meeting summaries to a Slack channel
- [ ] **Notion** — Enable to push transcripts or summaries to Notion pages
- [ ] **CRM** (Salesforce / HubSpot / Zoho / Pipedrive) — Connect and configure field mapping
- [ ] **ATS** (Greenhouse / Lever) — Connect if using for recruiting
- [ ] **Zapier / Make** — Set up automations if needed (e.g., create tasks from action items)
- [ ] **Dialers** (Dialpad / JustCall / Aircall) — Connect if running sales calls via dialer

---

## Phase 5: AI & Summary Configuration

- [ ] Review default AI summary template — adjust topic labels if needed
- [ ] Enable **Action Item Detection** (Settings → AI Features)
- [ ] Enable **Speaker Recognition** and label your own voice
- [ ] Add **Custom Vocabulary** for technical terms, product names, or industry jargon (improves accuracy)
- [ ] Configure keyword tracking (words you want flagged in transcripts)
- [ ] Set up **Smart Search** filters for your team's most common queries

---

## Phase 6: First Real Meeting

- [ ] Schedule a meeting — confirm Fred appears on the invite
- [ ] Run the meeting normally
- [ ] After the meeting, open the Fireflies dashboard
- [ ] Review the AI summary — does it capture what matters?
- [ ] Check the action items — are they correctly identified?
- [ ] Test the transcript search — search for a keyword from the call
- [ ] Share the summary with a teammate

---

## Phase 7: Ongoing Habits

- [ ] Review analytics weekly (talk time, topic trends, sentiment)
- [ ] Update custom vocabulary as new terms come up
- [ ] Star or bookmark important meetings for easy reference
- [ ] Use the mobile app for meetings on the go
- [ ] Explore conversation intelligence reports monthly (Business plan)

---

## Troubleshooting

| Problem | Fix |
|---|---|
| Bot didn't join the meeting | Check calendar connection; manually invite bot via email |
| Transcript is inaccurate | Add custom vocabulary; ensure good audio quality |
| Summaries missing key points | Check if AI summary was enabled; upgrade plan if on Free |
| Integration not syncing | Re-authenticate the integration; check API permissions |
| Bot joined but didn't record | Check recording permissions in Zoom/Teams settings |
