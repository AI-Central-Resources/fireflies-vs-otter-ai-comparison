# ✅ Otter.ai Setup & Onboarding Checklist

> Get Otter.ai up and running in under 10 minutes. It's one of the fastest AI meeting tools to configure and start using immediately.

**Tool Profile & Full Details:** [Otter.ai on AI Central Resources](https://www.aicentralresources.com/tool/otterai)

---

## Phase 1: Account Setup

- [ ] Sign up at [otter.ai](https://otter.ai) (use work email for team features)
- [ ] Verify your email address
- [ ] If you're a student/teacher, use your `.edu` email for 20% discount on Pro
- [ ] Choose your plan (Basic is free — no credit card required)
- [ ] Complete the onboarding walkthrough

---

## Phase 2: Meeting Platform Integration

- [ ] Connect your calendar:
  - [ ] **Google Calendar** — grant read access
  - [ ] **Outlook Calendar** — grant read access
- [ ] Install **OtterPilot** (the AI meeting bot) for your platforms:
  - [ ] **Zoom** — Install from Zoom App Marketplace
  - [ ] **Google Meet** — Enabled automatically via Google Calendar
  - [ ] **Microsoft Teams** — Install the Otter.ai Teams app
- [ ] Enable auto-join for all scheduled meetings (or configure per-meeting)
- [ ] Run a test call to confirm real-time transcription appears correctly

---

## Phase 3: Real-Time Transcription Setup

- [ ] Enable live captions in your preferred meeting platform
- [ ] Share your Otter.ai real-time link with meeting participants (optional)
- [ ] Set your speaker name for accurate speaker labels
- [ ] Add other frequent speakers' names to improve recognition
- [ ] Add **Custom Vocabulary** for technical jargon and product names

---

## Phase 4: CRM Integration (Sales Teams)

- [ ] Connect **HubSpot**:
  - [ ] Go to Settings → Integrations → HubSpot
  - [ ] Authenticate with HubSpot credentials
  - [ ] Map fields: action items → tasks, meeting notes → activity log
- [ ] Connect **Salesforce**:
  - [ ] Go to Settings → Integrations → Salesforce
  - [ ] Authenticate and authorize access
  - [ ] Configure which Salesforce objects to update (Contacts, Opportunities, etc.)
- [ ] Run a test call and verify data appears in CRM within 5 minutes of meeting end

---

## Phase 5: Other Integrations

- [ ] **Slack** — Connect to receive meeting summary notifications in a Slack channel
- [ ] **Google Docs** — Enable export to automatically save transcripts
- [ ] **Dropbox** — Connect for automatic backup of audio recordings
- [ ] **Zoom Cloud Recording** — Enable if you want Otter.ai to also transcribe your Zoom cloud recordings
- [ ] **Zapier** — Set up automation flows if needed

---

## Phase 6: AI Features Configuration

- [ ] Enable **Automated Summaries** (on by default — check it's working)
- [ ] Enable **Action Item Detection** and test with a real meeting
- [ ] Set up **AI Chat** — ask a question about a past meeting to confirm it works
- [ ] Configure **notification preferences** (email / Slack summary after each meeting)
- [ ] Set default sharing settings (private / team / everyone with link)

---

## Phase 7: Team Setup (Business Plan)

- [ ] Invite team members from Admin panel
- [ ] Create shared folders or channels for team meeting notes
- [ ] Configure admin permissions — who can see/edit whose meetings
- [ ] Set up billing admin and manage seats
- [ ] Review data retention settings

---

## Phase 8: First Real Meeting

- [ ] Join your next scheduled meeting — OtterPilot should join automatically
- [ ] Watch the real-time transcript scroll as people speak
- [ ] After the meeting, open the Otter.ai dashboard
- [ ] Review the automated summary and action items
- [ ] Test the **AI Chat** — ask "What were the key decisions?" or "What did [name] say about [topic]?"
- [ ] Check if CRM was updated (if HubSpot/Salesforce integrated)

---

## Phase 9: Mobile Setup

- [ ] Download the Otter.ai app (iOS or Android)
- [ ] Sign in with your account
- [ ] Test in-person transcription using the mobile mic (great for face-to-face meetings)
- [ ] Enable background recording (for meetings where no bot is present)

---

## Troubleshooting

| Problem | Fix |
|---|---|
| OtterPilot didn't join | Check calendar connection; invite bot manually via email |
| Real-time transcription delayed | Check internet speed; close other bandwidth-heavy apps |
| Transcript inaccurate | Add custom vocabulary; ensure clear audio and minimal background noise |
| CRM not updating | Re-authenticate integration; check field mapping settings |
| Transcription minutes used up | Upgrade plan or wait for monthly reset |
| AI Chat gives wrong answers | The meeting may not have been fully processed yet — wait a few minutes |
