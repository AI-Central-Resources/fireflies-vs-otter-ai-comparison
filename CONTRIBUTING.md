# 🤝 Contributing to This Repository

Thank you for helping keep this resource accurate and useful. This repo is maintained as a community reference for anyone comparing Fireflies.ai and Otter.ai.

---

## What We Welcome

- ✅ **Pricing updates** — both tools change pricing regularly; PRs with updated figures and a source link are always welcome
- ✅ **New integrations** — if you spot an integration we missed in the JSON files, add it
- ✅ **Accuracy corrections** — if a feature has changed, flag it via issue or fix it via PR
- ✅ **New use cases** — if your team has a workflow that isn't covered in `use-cases.md`, add it
- ✅ **New alternatives** — if a tool we haven't listed is worth knowing about, add it to `alternatives.md`
- ✅ **Checklist improvements** — onboarding steps that are outdated or missing

---

## What We Don't Accept

- ❌ Promotional or affiliate content without disclosure
- ❌ Content that isn't factually accurate or sourced
- ❌ Large structural rewrites without opening an issue first
- ❌ Additions to `data/*.json` without a source link in the PR description

---

## How to Contribute

1. **Fork** this repository
2. **Create a branch** with a descriptive name (`fix/otter-pricing-may2026`, `add/krisp-alternative`)
3. **Make your changes** — keep edits focused and minimal
4. **Submit a Pull Request** with:
   - A clear description of what changed
   - A source link (official pricing page, product changelog, etc.)
   - The date you verified the information

---

## Reporting Issues

If you spot something wrong but don't want to submit a PR, open an issue with:
- The file and section where the error appears
- What's incorrect
- What the correct information is (with a source if possible)

---

## Questions?

Open an issue and we'll respond.
